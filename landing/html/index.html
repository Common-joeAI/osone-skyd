How it works
Simple. Decentralized. Yours.

OSONE runs a lightweight agent on your device that donates idle compute to the network. In exchange, you get full access to the AI.

Step 01

Install the node agent

One command on Windows, Linux, Mac, or Android. The agent runs silently in the background.

Step 02

Contribute idle compute

When your device is idle, OSONE uses spare CPU/GPU cycles to run AI workloads for the network.

Step 03

Get free AI access

Use the OSONE AI — chat, automate, analyze — with no usage limits, no subscriptions, no ads.

Step 04

The network grows

More nodes = more compute = smarter, faster AI. Everyone benefits as the hive expands.

Install
Join the Hive in 60 seconds

Pick your platform. One command and you're in.

🪟

Windows

PowerShell — runs as a background scheduled task
$env:HIVE_TOKEN=""; $env:OSONE_URL="https://app.osone.org"; iex (irm https://app.osone.org/agent/install.ps1)
↑ Click to copy · Get your token at app.osone.org
🐧

Linux / macOS

bash — installs as a systemd service (Linux) or launchd (Mac)
HIVE_TOKEN="<your-token>" OSONE_URL="https://app.osone.org" bash <(curl -fsSL https://app.osone.org/agent/install.sh)
↑ Click to copy · Requires Python 3.8+
🤖

Android

Install via Termux or sideload the OSONE APK
  • Install Termux from F-Droid
  • Run: pkg install python && pip install websockets psutil
  • Then run the Linux install command above
📦 Termux (F-Droid) ⬇ node.py
🍎

iPhone / iPad

Run via the a-Shell or Pythonista app
  • Install a-Shell from the App Store
  • Run: pip install websockets psutil
  • Download node.py and run it — connects to the hive automatically
📱 a-Shell ⬇ node.py
